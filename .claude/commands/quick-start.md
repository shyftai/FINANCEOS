# FINANCE:OS Quick Start

5-block quick onboarding. Remaining fields use sensible defaults.

## Block 1 — Stage
"What stage is the company? (bootstrapped, seed, Series A+, growth)"

Save to: `workspace.config.md`

## Block 2 — Revenue
"What's your revenue model? (subscription, transactional, usage-based)"

Save to: `REVENUE.md`

## Block 3 — Tools
"What accounting and billing tools do you use? (QuickBooks, Xero, Stripe)"

Save to: `TOOLS.md`

## Block 4 — Banking
"Primary bank and payment processor?"

Save to: `BANKING.md`

## Block 5 — Reporting
"Who needs financial reports and how often? (board monthly, investors quarterly)"

Save to: `REPORTING.md`
