## v1.0.0 — Launch

### Core system
- Boot sequence with ASCII banner and system status
- Workspace architecture with financial sources of truth
- Role-based onboarding (Founder, CFO, Controller, Bookkeeper, Fractional CFO)
- MCP server and API key detection
- Solo/team collaboration mode
- Quality gates (7-point review)
- Learnings system (persistent financial intelligence)
- Financial roadmap with auto-generated to-dos

### Commands (22)
- Daily: today, dashboard
- Setup: onboard, connect
- Cash: cashflow, runway, forecast, scenario
- Revenue: revenue, invoices
- Expenses: expenses, vendors, burn, optimize
- Budget: budget, variance
- Tax: tax-prep, deadlines
- Report: report, pnl, balance, metrics
- Audit: reconcile, audit
- Agency: portfolio

### Integrations
- Accounting: QuickBooks, Xero
- Banking: Mercury, Brex, Wise, Plaid
- Payments: Stripe
- Payroll: Gusto, Rippling
- Invoicing: FreshBooks
- Expenses: Ramp, Expensify
- Scale: Claude API for batch processing
